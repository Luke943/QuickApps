An example UI created using QtDesigner. Has no real function.

Files:
- AdvancedUI.ui - file generated from QtDesigner
- containerUI.py - Python script gernerated from AdvancedUI.ui by pyuic5
- app.py - inherits UI from containerUI.py and adds functionality

Run app.py to see the app.

Convert .ui to .py from terminal by running:
pyuic5 inputfilename.ui -o outputfilename.py

Guide: https://www.youtube.com/watch?v=ksW59gYEl6Q&ab_channel=PyTutorials
